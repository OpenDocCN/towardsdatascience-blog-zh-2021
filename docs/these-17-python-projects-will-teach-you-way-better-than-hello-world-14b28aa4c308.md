# 使用标准 Python 或 Python 包的初学者的 5 个 Python 项目

这些面向初学者的 Python 项目直接来自源代码:他们使用普通的 Python，或者他们使用一些非常酷的 Python 包。这些面向初学者的 Python 项目将包含有用的文档、评论以及其他尝试过并喜欢这些项目的 Python 初学者的经验。

## [1。用“Tweepy”做一个推特机器人](https://realpython.com/twitter-bot-python-tweepy/)

痴迷于社交媒体？我也是。我关注了几个机器人(比如[这个搞笑的机器人](https://twitter.com/kauIhiIoshudon)是基于我读过的一本很棒的书)，并且总是想知道如何制作一个。

虽然[这个教程](https://realpython.com/twitter-bot-python-tweepy/)对于 Python 初学者来说听起来很高级，但它也能让你稍微沉浸在自己的激情中，也许还能让你的幽默感得到提升。本教程将引导您完成 5 个步骤:安装“Tweepy”包，注册成为 Twitter 开发人员以使用其 API，使用 Tweepy 调用 Twitter API，构建 Twitter 机器人，以及使用 Docker 和 AWS 将机器人部署到服务器。

是的，工作量很大！但它会很有趣，会教你很多技能，而且足够有趣，让你度过难关。

## [2。写一个你自己的冒险脚本](https://coding-grace-guide.readthedocs.io/en/latest/guide/lessonplans/beginners-python-text-based-adventure.html)

当我还是个孩子的时候，我曾经很喜欢这些书，当我得知你可以使用 Python 来创建你自己的书时，我真的很兴奋！[这个教程](https://coding-grace-guide.readthedocs.io/en/latest/guide/lessonplans/beginners-python-text-based-adventure.html)很棒，因为它贯穿了你需要的每一个步骤，并且解释了你将要使用的很多背景代码。

这也很棒，因为最终，你会用自己的双手创造出你真正热爱的东西。你可以在网上与朋友、家人和陌生人分享。让您的创造力接管并使用这些 Python 项目来帮助初学者学习基础知识。

## [3。创建博客](https://djangocentral.com/building-a-blog-application-with-django/)

嘿，你在博客上看这个！你猜怎么着？你可以结合使用 Python 和 Django 来创建一个博客。

本教程将教你如何用 Django 构建一个博客应用程序，让用户创建、编辑和删除文章。你将有一个列出你所有博客文章的主页，以及一个针对每篇文章的详细页面。

这使用了一个叫做 Django 的公共 Python 框架。Django 是一个用 Python 编写的开源 web 框架。框架基本上是一种捷径。你可以用纯 Python 来做这件事，但是 Django 会让它简单得多。再说一次，这个教程真的很好，因为它不只是给出模糊的指示，它给你一步一步的完整演示来设置你的环境，谈论数据库模型，并实际创建博客。

如果你有话要说，需要一个地方来说，这个项目是完美的。作为奖励，你可以使用你的 Python 博客来打动潜在的雇主，获得你梦想中的 Python 工作。

## 4.制作一个登录系统，[第一部分](https://www.youtube.com/watch?v=RHu3mQodroM)和[第二部分](https://www.youtube.com/watch?v=SUC1aTu092w&t=0s)

这个面向初学者的 Python 项目对于任何想成为未来 web 开发人员的人来说都是非常棒的。每个好的网站都需要一个登录系统，好消息是你可以用 Python 做到这一点。这个选项不是一个热情的项目，而是一个实际的项目，但它会教你很多你需要的技能，结果是一个真正有效的交互系统。

而不是一个书面的教程，你应该看看这两个 YouTube 视频来学习如何制作一个登录系统。他们会带你经历你需要什么代码的实际步骤，以及解释你所做的东西背后的概念的更理论的步骤。

[第一部](https://www.youtube.com/watch?v=RHu3mQodroM)长约 25 分钟，而[第二部](https://www.youtube.com/watch?v=SUC1aTu092w&t=0s)长 32 分钟。

## [5。将 Python 连接到 StackOverflow](https://pypi.org/project/howdoi/)

我认为这个选项不仅仅是 Python 初学者的项目，更应该是每个 Python 用户的需求。

关于编码，我学到的一点是，你很少会疯狂地敲键盘，写出一行行代码，然后神奇地填满你的屏幕。相反，它看起来更像是从互联网上复制零碎的代码，并在 StackOverflow 上搜索它，当它没有做它应该做的事情时。

这就是为什么我喜欢[这个教程](https://pypi.org/project/howdoi/)，它是基于一个叫做`howdoi`的包。正确设置后，它会从命令行为您提供即时的编码答案。理想情况下，这意味着你不必打开浏览器来寻求帮助。如果你非常有野心，一旦你设置好了，你可以更进一步，创建一个 GUI 或者 flask web 应用来包装`howdoi`包。

一旦你意识到你花了多少时间在谷歌上，你就会意识到把它作为初学者的 Python 项目的价值。

# 使用自己的数据为初学者设计的 5 个 Python 项目

什么是项目，而只是一个需要数据来回答的问题？如果上面 12 个针对初学者的 Python 项目没有一个让你感兴趣，那么好消息是，你可以使用的项目数量是无限的。你所需要的只是一撮创造力和想要知道更多的激情。

有了今天的互联网，你可以很容易地从你感兴趣的任何来源找到数据，并创建你自己的项目。这将比上面的选项更难，因为没有护栏，但当你最终找到你的答案时，也可能会更令人满意。

这里有五个很好的互联网数据源，可以用来为初学者创建自己的基本 Python 项目。请将这些简单的 Python 项目视为一个起点。

## [13。538](https://data.fivethirtyeight.com/)

Nate Silver 最近经常被扣篮，但好消息是他的网站 538 仍然是一个很酷的地方，可以找到简单的初级 Python 项目。只要看看[这个网站](https://data.fivethirtyeight.com/)，向下滚动列表，看看有没有你感兴趣的东西。

它在不断更新，所以现在我正在看俱乐部足球预测、MLB 预测、政治预测等等。自己看一看，看看你想潜入什么。看看 538 如何运行他们的分析，并做你自己的旁边。

## 14。整洁的星期二

这有点厚脸皮，因为这些编程项目最初是为 R 而不是 Python 设计的。但它们都是数据科学的优秀语言。Tidy Tuesday GitHub repo 每周二都有新的数据，还有一些有趣的想法，适合初学 Python 的项目和可视化的想法，你可以借用一下。

过去几周的例子包括运输成本、艺术收藏、肯尼亚人口普查、财富和收入等等。让你自己被这些数据甚至其他人围绕这些数据所做的项目所鼓舞。

## 15。Buzzfeed

是的，我知道，他们不只是做小测验！[他们的 GitHub repo](https://github.com/BuzzFeedNews) 是开源数据、分析、库、工具和指南的另一个奇妙来源。看看吧，看看你感兴趣的，开始分析吧！

他们查看 H2 签证认证数据、枪支背景调查、政治捐款、养老院数据等等。

## [16。美国宇航局](https://pds.nasa.gov/datasearch/data-search/)

太空是我们这个时代最大的谜团之一。如果你曾经梦想成为一名宇航员，像我一样，这可能是你调查数据的地方。美国宇航局运行的行星数据系统提供了基于大量不同目标的数据，你可以选择这些目标，包括彗星、小行星、所有的行星，甚至太阳风。

这个潜在的项目对于 Python 初学者来说可能是一个挑战，但是如果这是你感兴趣的那种数据，这是值得的。

## [17。AWS](https://github.com/awslabs/open-data-registry/)

任何人都不应该对 AWS 拥有绝对庞大的开放数据注册库感到惊讶。他们有来自艾伦人工智能研究所、数字地球非洲、脸书永久数据的数据集，还有更多来自美国宇航局的东西，甚至还有亚马逊可持续发展数据倡议。

自然，AWS 并不拥有这些数据——他们只是为选择公开这些数据的各种政府组织、研究人员、企业和个人托管这些数据。他们的 GitHub repo 也包含一些如何使用数据的例子。

随便浏览一下，随便逛逛，找到你觉得有意思的。
