# 1.sci kit-学习:

![](img/70d501e053ad474b7d6f28839f8f0a97.png)

照片由[杰斯·贝利](https://unsplash.com/@jessbaileydesigns?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

Scikit-learn 是绝对最好的机器学习库之一。它是以最好的方式构建的，这使得它可以与其他优秀的数据科学库兼容，如 Numpy、Pandas 和 matplotlib。

使用 scikit-learn 来实现多个机器学习项目的结果通常是一个好主意。它提供了用户开始机器学习所需的几乎所有奇妙的工具。

所有基本的机器学习算法，如 K 近邻、线性回归、逻辑回归、随机森林、决策树和其他类似的库都是预先构建的，允许用户相应地计算、评估、训练和适应他们的机器学习项目。

但是这些很难说是 sklearn 最好的部分，因为它为用户提供了如此多的工作，理解如何最好地利用这一官方文档变得非常重要。

## 怎么用？

为了确保 sklearn 在稳定的学习过程中发挥最佳作用，我首先建议您自己从头开始实现大多数机器学习算法。如果你被困在某个地方，你可以利用他们的类和模块来温习你的概念和代码。

一旦您对基本概念有了基本的理解，您就可以相应地使用它们的许多类、函数和应用程序了。为了 sklearn 的完美效用，利用他们的[官网](https://scikit-learn.org/stable/)，了解每个函数的用途和各自的参数。

sklearn 模块提供了许多选项来解决各种任务，例如拆分数据、创建和操作数组和数据元素、拟合和训练模型等等。

为了清楚地理解本文档，最好的方法是阅读基本和重要的参数，并尝试自己解决他们网站上提供的给定示例。最好是同时编码并相应地检查结果。

# 3.PyTorch:

![](img/cf1aa0465cf8ac79a1401f62b9873db2.png)

照片由[乔纳森派](https://unsplash.com/@r3dmax?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

PyTorch 由脸书人工智能研究实验室(FAIR)的团队开发，是最好的深度学习库之一，可以执行大量的项目。由于 PyTorch 的代码取向是高度 Pythonic 化的，所以越来越多的开发人员开始倾向于使用这个深度学习库来完成各种任务。

PyTorch 不仅以其简单的编码过程为 TensorFlow 提供了强有力的竞争，而且还具有更简单的安装过程、易用性和更基本的精彩文档，初学者开发者可以从这些文档中轻松获得有用的信息。

## 怎么用？

利用 PyTorch [文档](https://pytorch.org/docs/stable/index.html)的最佳方式是快速掌握一些更基本的东西，如 Torch.nn、torch.nn.functional 和 torch。张量功能，以及适应一些 CUDA 和设备设置，你可能需要为某些实施。

一旦你对他们的文档的基本要素有了基本的理解，我强烈推荐你继续阅读他们最好的部分。PyTorch 官方文档中的教程部分几乎涵盖了初学者可能想要学习和适应的所有基本主题和概念。

# 4.Keras:

![](img/14f2d032c6dd25057f28f38aad0bec07.png)

由[道格拉斯·洛佩斯](https://unsplash.com/@douglasamarelo?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 拍摄的照片

Keras 是最好的高级库之一，可供开发人员充分利用并成功完成各种项目。Keras 现在与 TensorFlow 2.0 集成在一起，它们基本上是同一个组件。

在 Keras 的帮助下，你可以像 TensorFlow 一样执行众多任务，但代码流会简单一点。需要您执行更多功能和自定义类来解决特定任务的操作可以用预定义的代码替换。

相比之下，初学者的 Keras 代码实现比类似代码的 TensorFlow 实现简单。由于 Keras 现在被合并到 TensorFlow 2.0 中，因此更容易混合使用这两个惊人的库来执行几乎任何类型的深度学习和神经网络任务。

## 怎么用？

Keras 库的官方文档可以通过下面的[链接](https://keras.io/)获得。要学习 Keras 并最大限度地利用它，建议查看指南并简要了解该库提供的所有选项。

虽然不可能掌握 Keras 中的每一个 API 引用、类或函数，但了解一些基本层、损失函数、模型架构和其他类似的功能将会有很大帮助。它还有助于理解这些功能中集成的一些基本参数。

一旦您掌握了几乎将在每个项目中使用的一些基础知识，例如 Keras 层和 Keras 损失，您现在可以继续查看一些在其文档的示例部分中提供的惊人的示例项目。有很多很酷的项目，你可以探索并尝试获得，以及复制结果。

由于 TensorFlow 和 Keras 合并为一个单元，因此同时使用第二点和第四点的技巧是一个好主意。
