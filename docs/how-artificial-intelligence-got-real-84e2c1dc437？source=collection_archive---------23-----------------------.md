# 人工智能是如何成为现实的

> 原文：<https://towardsdatascience.com/how-artificial-intelligence-got-real-84e2c1dc437?source=collection_archive---------23----------------------->

很长一段时间以来，人工智能是一种思想实验——一种科幻比喻。感觉就像一眨眼的功夫，它现在在我们的日常生活中无处不在。我们是怎么到这里的？

一个很好的起点是[的 Mike Ferguson](https://medium.com/u/2f98f6dd4deb?source=post_page-----84e2c1dc437--------------------------------) 关于人工通用智能的新[系列，他耐心地带领我们了解几十年来哲学、神经科学和相邻学科对人类创造 AGI 的潜力的理论。(当你完成介绍后，你已经可以继续第二部分](/what-is-artificial-general-intelligence-4b2a4ab31180)[和第三部分](/the-cognitive-science-of-agi-139e1f4b31d5)了。)

还有心情阅读一些高水平的基础读物吗？跟随 Mike 的帖子，听听 Gabriel de Longeaux 对信息论的介绍，这是他专门为数据科学家量身定制的。

如果你被人工智能更实际、更现实的方面所吸引，你不会想错过阿曼达·林登对塑造人工智能产品的设计过程的清晰观察。我们经常(太经常？)将人工智能视为一个以开发者为中心的领域，但阿曼达认为，人们是通过设计来体验它的好处和力量的。

与此同时，[凯西·科兹尔科夫](https://medium.com/u/2fccb851bb5e?source=post_page-----84e2c1dc437--------------------------------)提醒我们[没有一个人工智能的回报是无风险的](/the-ai-reliability-paradox-ac8d1739206b)——尤其是当你认为你已经抓住了它，并且你创造的系统已经好得不能再好了。卡西警告说:“基于数学和数据的解决方案最危险的一点是，非专家过于相信它们。不要成为那种在复杂的任务上依赖完美的坐以待毙的人。”

非常不同的挑战[在最近的 TDS 播客节目](/the-surprising-challenges-of-global-ai-philanthropy-7a37718f9679)中浮出水面，主持人 [Jeremie Harris](https://medium.com/u/59564831d1eb?source=post_page-----84e2c1dc437--------------------------------) 与嘉宾李岩讨论了在慈善领域使用人工智能的局限性。作为非营利组织 Techies Without Borders 的联合创始人，她亲身经历了在边缘化社区和资源匮乏地区实现人工智能承诺的困难。

数据清理是人工智能可能在不久的将来让数据分析师的生活变得更加轻松的另一个领域，但我们仍然没有完全实现。与此同时， [Randy Au](https://medium.com/u/5bd01667d4e5?source=post_page-----84e2c1dc437--------------------------------) 坚持认为，目前从事这项工作的数据科学家[不要再把它当作无聊的杂务，而是把它](/data-cleaning-is-analysis-not-grunt-work-edb386f9429d)当作“在进行全面数据分析的道路上的一系列可重用数据转换*”*
