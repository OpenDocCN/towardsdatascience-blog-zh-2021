随着我越来越深入数据科学的世界，出现了大量关于高级机器学习主题的文章和教程。至少有一大部分细节和教程是关于机器学习如何工作以及什么库最适合使用的。然而，我注意到，对于那些刚刚起步的人来说，并没有太多的机会。所以，这就是本文的目标。我们将从头开始，学习最基本的知识。首先，我们将讨论什么是机器学习以及它为什么重要的简要概述，然后我们将看看机器学习的三种类型。

# **什么是机器学习**

机器学习是计算机科学和人工智能(AI)的结合。这种组合使用复杂的计算和问题解决方法，创建并遵循模式来做出决策。做出这些决定是为了模仿人类的思维方式，随着时间的推移，这将改进模型和决策过程。

随着大数据的不断扩大，数据科学的重要性和对机器学习的需求也在增加。机器学习很重要，因为它可以用来帮助更快地做出决策和进行数据分析，特别是在大数据方面。目的是分类和预测趋势，这是数据挖掘项目的关键。如果你想更多地了解为什么使用机器学习，我建议做更多关于数据科学和机器学习用途的研究。但是对于本文来说，背景信息就足够了。让我们来看看不同类型的机器学习。

# **类型 1:监督学习**

我们将讨论的第一种机器学习是监督学习。在这种方法中，您从较大的数据集中抽取样本。此示例用于表示可以从数据中推断出的相关性和关系。基本上，它将试图总结不同的情况，以了解可以做出什么预测或如何对数据进行分类。在经过样本集之后，真正的数据集将被消化，机器将在分类或预测方面完成剩余的工作。但是，不要把从样本中获得的知识当成唯一的知识。在监督学习期间，收集的每一条数据都是出现差异的机会或肯定当前系统的机会。基本上使用初始训练集，但是系统从不停止学习。因此，它工作得越好，通过它的数据越多，它就会变得越好。

现实世界中用于预测的监督学习的一个例子是，如果你根据故事的描述来猜测故事的结局。然而，在你得到数据之前，你会得到相似的故事以及这些故事的结局。基于你从例子中收集到的信息，你可以用你现有的知识进行预测，但是你可以在进行中学习做更多的预测。对于分类来说，就像看一看附有描述的照片一样。根据照片的外观和描述，您可以为这些照片设置分类。一旦你对这些照片进行了分类，你会得到一套没有描述的照片，并根据你已经知道的对它们进行分类，这也让你可以边学边做。

简单想一下，监督学习下的两种学习类型是分类和回归。分类很像我们照片例子中的分类。创建的这类类别是非真实值，这只是意味着这些值无法具体衡量。一个例子是基于颜色的分类，例如将纱线分为“绿色”、“黄色”或“红色”。第二种是回归。这涉及到任何真实的价值，是可以衡量的。例如，如果你使用一个网站来查看房屋列表，你可能有一个你想要保持的价格范围。回归分析会根据价格范围对房源进行分类，所以只给出你预算内的房子。

# **类型 2:无监督学习**

可以猜到，无监督学习是有监督学习的反义词。可能会提供相同的数据，但没有样本集可供学习。也没有描述来帮助确定数据可以分成什么类别。这意味着不需要人工干预，不像监督学习。相反，数据被提供给系统，它使系统抽象地找到数据点之间的关系。这些系统结构是隐藏的，可以适应数据。由于这种适应性，无监督学习能够开发更多的后部署。

举个例子，无监督学习就像给你一个要打扫的壁橱。在没有任何信息的情况下，你会根据观察的相似性来比较项目。随着你的发展，你会形成自己的组织结构。然而，项目，后来可能会让你第二次猜测这个结构，并因此采用一个新的结构，因为你发现新的变化。

正如我们之前所做的，两种类型的无监督学习模型是聚类和关联。聚类基本上是基于感知的相似性将数据点聚集在一起。关联是寻找将数据点链接或关联在一起的规则。基于它们的定义，两者看起来非常相似。澄清一下，聚类就像成簇，其中您创建组来对数据进行分类，而通过关联，您会发现更多的规则将数据链接在一起，并对它们进行专门分组。

# **半监督学习**

所以不是另一种类型，但我认为应该提到介于两者之间的半监督机器学习。当您需要一些结构，但没有时间标记所有内容时，可以标记部分信息，这样您就可以对结构进行一些控制，而且还可以灵活地让机器自己做出决策。对于许多现实世界中的机器学习问题，半监督是在消耗宝贵时间的同时进行一些监督的现实答案。

# **类型 3:强化学习**

我们今天要看的最后一种机器学习模型是强化学习。强化学习是一种类似于监督学习的机器学习模型，但没有样本数据集。相反，机器是经过训练的，就像你训练你的宠物一样。对机器进行时间敏感的强化，要么肯定所做的决定，要么否定猜测的“正确性”。像监督学习模型一样，有一个感知的正确和错误的方法来处理数据。然而，钢筋使用试错法计算。

虽然你可能不需要一个强化的例子，但是我们会想到一个快速的例子。当你训练你的狗的基本命令，如坐和停留，狗不能理解你的话。然而，当狗执行适当的任务，你给他一个奖励。然后，当你说“坐下”时，或者当它得到这个数据时，它就知道要做“坐下”的动作，这样它就得到肯定它所做的是正确的奖励。你的机器将以同样的方式被训练，在对数据做出正确假设的基础上被给予一个正的返回变量。

为了强化，这个机器学习模型使用正负强化。使用的两种强化类型是马尔可夫决策过程和 Q 学习。通常，这是我描述每个是什么的地方。然而，这些类型有些复杂，而且篇幅太长，无法包含在这个简要的概述中。稍后，我可能会写更多关于这两种模型的内容，但是现在，这里已经有一些作者专门撰写了一些文章来理解这两种模型。

# **结论**

随着大数据变得越来越重要，机器学习也是如此。这是因为随着人们越来越依赖数据科学，人们正在寻求更快的决策方法。机器学习就是这样一个过程。

在本文中，我们讨论了三种不同类型的机器学习:监督学习、非监督学习和强化学习。我们还简要地看了使用该学习模型的描述、例子和方法类型。虽然没有同样多的细节，但我们也快速看了一下半监督学习，这是监督和非监督学习的结合，就像它听起来一样。

无论你决定使用什么库，你为你的机器学习项目选择的模型都会影响它的编写方式。我们讨论的所有模型都有优点和缺点，但它们仍然是具有有益效果的强有力的选择。我希望你学到了更多关于机器学习的知识，并且喜欢这次对机器学习类型的探究。下次见，干杯！
