# 数学动画，不可复制的研究，用数据讲故事

> 原文：<https://towardsdatascience.com/math-animations-irreproducible-research-and-telling-stories-with-data-d9b55aa3887b?source=collection_archive---------23----------------------->

“故事”是一个有时感觉被过度使用的词，包括在数据科学的背景下。不是每一个结构清晰、有用的幻灯片都是一个故事，这没关系。但是正如[玛丽·勒费夫尔](https://medium.com/u/2a04bf49928f?source=post_page-----d9b55aa3887b--------------------------------)在[她的关于令人信服的数据叙事](/how-to-tell-a-compelling-story-with-data-d00ea8cd1bb)的帖子中所言，*有*种经过时间考验的方法可以让任何分析令人难忘和引人入胜，那么为什么要满足于枯燥和敷衍呢？

TDS 作者*当然*不会安于干巴巴的敷衍了事；我们经常惊讶于我们分享的故事和声音的范围。给乍看上去技术性很强的话题注入活力需要高超的技巧和巨大的努力。当你听到“可重复研究”时，你会直接跑到你的厨房把一些爆米花扔进微波炉吗？也许你应该，如果你议程上的下一个项目是 [Vincent Vanhoucke](https://medium.com/u/2879ca55026a?source=post_page-----d9b55aa3887b--------------------------------) 呼吁[认识到*不可复制的*研究](/i-irreproducible-research-dbc48eb140ac)的价值，基于他在谷歌的机器人研究工作。

当你的爆米花还新鲜的时候，跳到[阿诺·古兹曼-安内斯](https://medium.com/u/4ac115d4b44f?source=post_page-----d9b55aa3887b--------------------------------)对疫情期间[航空机队优化的探索，一个他清晰而耐心处理的复杂问题。](/fleet-management-during-the-pandemic-an-optimization-problem-6a2572273c6f)

你还剩下几粒果仁吗？你不会后悔进入 [Isabelle Augenstein](https://medium.com/u/10defa25fcd9?source=post_page-----d9b55aa3887b--------------------------------) 对[NLP 学者的维基百科条目](/who-is-wikipedia-famous-within-natural-language-processing-fa0c8e91bdf6)的研究，其中也讨论了这些维基页面告诉我们的关于该领域的偏见和代表性。

教另一个人如何完成一个具体的目标也是一种讲故事的形式，最好的老师会让他们的学生忘记他们正在学习。想把你的牙齿埋进一个吸引人的、有趣的教程吗？从 [Khuyen Tran](https://medium.com/u/84a02493194a?source=post_page-----d9b55aa3887b--------------------------------) 的指南开始，[展示了如何用 Python](/how-to-create-mathematical-animations-like-3blue1brown-using-python-f571fb9da3d1) 创建数学动画，然后继续进行 [Vishesh Khemani，博士](https://medium.com/u/92a54b8269fc?source=post_page-----d9b55aa3887b--------------------------------)对搜索算法的新尝试，[将抽象概念推入物理世界](/wheres-waldo-3dcd3c3b199f)(剧透提示:*书中涉及*)。

用 TDS editor [Elliot Gunn](https://medium.com/u/aad1101621dd?source=post_page-----d9b55aa3887b--------------------------------) 关于文档重要性的资源丰富的[专题](/the-last-mile-in-shipping-data-science-projects-well-c4fb73c35af3)来结束你的学习课程——它包括了我们在这个至关重要但经常被忽视的主题上发表的一些最好的文章。

如果你对一个引人入胜的故事的想法倾向于投机和元，不要担心:你不会空手而归。你可以花接下来的 11 分钟思考马克·萨鲁菲姆关于拉格朗日力学的想法，利用机器学习和物理学之间的联系。或者带上你的耳机，去悠闲地散步，听听 [Jeremie Harris](https://medium.com/u/59564831d1eb?source=post_page-----d9b55aa3887b--------------------------------) 在 TDS 播客上与 [Melanie Mitchell 的对话，他们讨论了人工智能](/existential-risk-from-ai-a-skeptical-perspective-35f0cd7c9fa4)的局限性，以及为什么我们都为人工智能霸主服务的未来可能不会*就在眼前*。
