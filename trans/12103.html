什么是 Python 集合

Python 集合是一种数据结构，用于存储唯一元素的无序集合。集合本身是可变的，但由不可变数据类型的元素组成(如整数、浮点、布尔、字符串、元组等)。集合不能由可变数据类型的元素组成(比如 list 、 dictionary 等等)。
