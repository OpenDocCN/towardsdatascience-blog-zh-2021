Altair安装和数据集介绍

您可以像安装任何其他Python库一样安装Altair。我们将它安装在一个名为altair_env的独立虚拟环境中，基于Python版本。

Numpy和Pandas是Altair的依赖，不用手动安装。如果您使用的是Anaconda，下面一组shell命令将设置环境，激活它，并安装所有需要的东西:

conda create — name altair_env python=3.8
conda activate altair_envconda install -c conda-forge altair altair_saver
conda install -c anaconda pandas-datareader
conda install jupyter jupyterlab

厉害！您现在可以启动Jupyter Lab并创建一个新的空笔记本。
