如何使用Q型鳍

Q-Fin发布到PyPi，可以用pip包管理器安装。

pip install qfin

安装后， qfin 可以与所需的定价模块一起导入。

当实例化 BlackScholesCall 或 BlackScholesPut 的对象时，会自动生成价格和希腊值…
