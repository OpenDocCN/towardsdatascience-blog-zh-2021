什么是Kats？

Kats 是一个轻量级的、易于使用的、通用的框架，用Python来执行时间序列分析，由脸书研究院开发。您可以将Kats视为Python中时间序列分析的一站式商店。

要安装Kats，请键入:

pip install --upgrade pip
pip install kats==0.1 ax-platform==0.2.3 statsmodels==0.12.2
