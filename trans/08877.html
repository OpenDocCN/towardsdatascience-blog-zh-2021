最流行的Python探索性数据分析库。

GitHub中有超过77k颗星星， Pandas-Profiling 是我们列表中最受欢迎的探索性数据分析工具。它易于安装，使用简单，结果无可挑剔。

您可以使用PyPI或Conda来安装Pandas-Profiling。

pip install pandas-profiling
# conda install -c conda-forge pandas-profiling

该安装允许您在终端窗口中使用pandas-profiling CLI。几秒钟之内，它会生成一个HTML报告，其中包含大量关于数据集的分析。

灵活的EDA操场。

如果你可以多眨几下眼睛，但需要更多地控制你的分析，这就是你需要的。 Pandas GUI 为您的数据框创建图形包装。您可以使用方便的接口，而不是编写代码。熊猫GUI与其说是一个快速探索工具，不如说是一个探索游乐场。

您可以用PyPI安装它:

pip install pandasgui

和Sweetviz一样，Pandas GUI也没有CLI。尽管启动它并不复杂，但是如果您不是Python用户，下面的CLI包装器可以帮助您。

结论
