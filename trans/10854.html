这一行代码在我的代码中总是节省了几行。⏳

pyforest就是解决方案！！

pyforest是一个python包，它在一行中导入了所有常用的包。它不是预装的，因此只有第一次使用时，你需要安装它。

pip install pyforest

与其将每个库都导入到你的程序中，不如写在一行下面，这样你就可以避免所有思考和导入所需库的麻烦。

import pyforest

导入pyforest后，pd.DataFrame()、np.arange()、sns.countplot()、sys.exit()等常用Python包中的所有函数都可以直接使用。

dir(pyforest)会给你pyforest导入的 105 包的完整列表。🍀
